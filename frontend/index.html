為學生與教師打造的線上物聯網實驗與學習平台

AI物聯網實驗室是免費、開源的線上學習平台:在瀏覽器裡即可撰寫並執行 Arduino、ESP32、Raspberry Pi Pico 等 19 種開發板的真實韌體,搭配 100+ 互動電子元件與 SPICE 精度的類比電路模擬。內建為初中、高中設計的互動課程與選擇題測驗,教師可建立班級、追蹤學生進度與成績。AI 智慧助教能自動搭建電路、解說程式。免安裝、免帳號即可開始。

主要功能

  • 真實 AVR8 模擬:Arduino Uno / Nano / Mega 2560 / ATtiny85(16 MHz,avr8js)
  • ESP32 / ESP32-S3 / ESP32-C3(QEMU)與 Raspberry Pi Pico(rp2040js)模擬
  • 互動課程:Arduino 入門、ESP32 物聯網 — 每課附電路範例一鍵載入與測驗
  • 師生班級管理:班級代碼加入、進度追蹤、成績報表
  • AI 智慧助教:用中文對話即可搭電路、寫程式、除錯
  • SPICE 類比電路模擬(ngspice-WASM):電阻、電容、電晶體、運算放大器
  • 示波器、電壓表、電流表、訊號產生器等虛擬儀器
  • Monaco 程式編輯器、序列埠監控視窗、Arduino 函式庫管理員

常見問題

AI物聯網實驗室是免費的嗎?
是。平台以 GNU AGPLv3 授權開源,完全免費,亦可自行架設於校內伺服器。
需要安裝軟體或購買硬體嗎?
不需要。所有模擬都在瀏覽器內執行,沒有實體開發板也能完成課程與實驗。
教師如何管理班級?
教師註冊帳號後可建立班級,學生輸入班級代碼加入;教師儀表板即時顯示每位學生的課程進度與測驗成績。
